使用 Topic 构建内容

 

        Topic 是具体文档内容（文本、图像、多媒体、链接等）的容器，通过增加、编辑、排版、删除 topic 的内容实现对整个文档的构建。可以创建不同的文件夹来管理多个 topic。

创建文件夹

        可以在 Contents 面板的根目录下创建文件夹，也可以在文件夹创建子文件夹。 

        在打开的项目中创建根目录文件夹：

  1. 在 Author 工具栏中单击 Contents

  2. 在 Contents 面板中，单击面板右上角的，然后单击 New Folder

  3. 在 Folder Details 对话框中，在“ 名称”中指定文件夹  名称。

  4. 点击 Create

创建 Topic

          可以在文件夹内或 Contents 面板的根级别上创建主题。
在文件夹中创建主题：
1.    在“作者”工具栏中，选择“ 目录”。“ 内容”面板将打开。
2.    在“ 内容”面板中，执行下列操作之一：
•    选择所需的文件夹，然后单击    面板的右上角，然后单击“  新建主题”。
•    右键单击该文件夹或单击该文件夹    ，然后选择  新建>新建主题。
注意：
如果不选择文件夹，则在“ 内容”面板的根级别创建主题  。 
3.    在“ 新建主题”  对话框中，根据需要在不同部分中指定主题的属性。
“  新主题”对话框与“ 主题属性”对话框具有相同的部分，但“文件信息”部分专用于主题属性。有关更多信息，请参见主题属性。
4.    点击创建。
该主题在“ 内容”面板中的选定文件夹内创建。您可以在文件夹或子文件夹内创建多个主题。
