下载与安装 

 

了解如何通过 DVD 安装应用程序，如何从 Adobe.com 下载应用程序，以及如何查找资源以对错误代码进行故障诊断。

环境要求

注意：本产品不支持 Mac OS 操作系统

  1. 酷睿 i3 及以上处理器

  2. Microsoft Windows 7、8.1 或 10

  3. 建议使用 4 GB 及以上的 RAM

  4. 安装需要 3 GB 的硬盘空间，安装期间需要额外的空间（不能安装在使用区分大小写的文件系统的卷上或可移动闪存存储设备上）

  5. 64 位 CPU 和操作系统

 

支持软件

  • Microsoft® Word 2016, 2013, 2010

  • Adobe Acrobat Pro DC (2015 release)、Pro 2017

  • Adobe Captivate 2019*

  • Microsoft SharePoint Online*

 

支持语言

RoboHelp（2019版）提供以下语言版本：

  • 英语

  • 法语

  • 德语

  • 日语

从 Adobe.com 下载并安装

注意：本产品不支持 Mac OS 操作系统

  1. 从 Adobe 网站打开已下载的安装程序文件

  2. 导航到产品文件夹中的 Set-up.exe

  3. 双击 Set-up.exe 以运行该程序

  4. 安装程序窗口打开后，如果未登录，使用 Adobe ID 登录

  5. 按照安装提示进行操作，直到安装完成
