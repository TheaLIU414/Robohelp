Skins、Lables 和 Master Pages

 

本章节介绍如何使用 Adobe RoboHelp 的 Skins、Lables 和 Master Pages 进阶功能。

使用 Skins

什么是 Skins

RoboHelp 中的 skins 用于在发布帮助系统之前更改和自定义帮助系统的外观。 Skins 可以通过更改输出的颜色，按钮，字体，图标，徽标和背景来自定义外观。 无论要发布的是响应式 HTML5 还是 WebHelp，RoboHelp 都会在发布窗口的“外观”面板中提供多个预定义的外观。这些预定义外观可以按原样应用，也可以对其进行编辑以实现首选的输出布局。

添加 Skins

点击标准工具栏左侧的 Output 选项卡，选择左侧的 skins 即可添加 skin。

编辑 Skins

添加或导入 skin 后，可以对布局、主页面和话题页面进行编辑。

重命名 Skins

选中需要重新命名的 skin，右键选择 Rename 或者按 F2。

升级 Skins

升级 RoboHelp Classic 项目时，RoboHelp 会自动升级所有响应式 HTML5 外观。 在 RoboHelp Classic 编辑器之外自定义的外观，请在 RoboHelp 2019 的布局面板中使用组件添加自定义 JavaScript 或 CSS 文件。进入 skin 的编辑界面，选择 Layout 中的 User Assets 即可添加自定义 JavaScript 或 CSS 文件。

导入 Skins

点击标准工具栏左侧的 Output 选项卡，点击 + 后选择 Import 即可导入 skin 文件（.skz 或 .slz 格式）。

导出 Skins

选中需要导出的 skin，右键选择 Export 即可导出 skin 文件到指定的文件夹。

删除 Skins

选中需要删除的 skin，右键选择 Delete 后，在弹出的确认框中选择 Yes 即可删除相应 skin 。

查找 Skins 文件的本机位置

选中需要查找的 skin，右键选择 Show in Explorer 即可查找 Skin 文件的本机位置。
