Robohelp 2019 介绍

 

Adobe RoboHelp（2019）是一款用于创建和发布内容的工具，它功能强大且操作直观，可以帮助您轻松高效地开发帮助系统。通过直观易用的功能你可以用最少的精力为不同的受众和各种设备提供定制的内容。RoboHelp 包含强大的内容编辑、布局创建、格式设置和搜索功能，可帮助你快速创建并快捷高效的管理内容。

RoboHelp 提供现代化的响应式 HTML5 和设备友好型输出预设，来创建流畅且个性化的内容。使用条件标签和表达式可以用多种格式针对不同的屏幕创建交互式和经过过滤的内容。您可以迎合更广泛或更有针对性的受众。

RoboHelp 2019 入门

主要功能

响应式 HTML5 和 CSS3 创作支持

借助现成的 HTML5 和 CSS3 创作支持，使用文本和多媒体创建高度沉浸式内容体验。轻松添加来自 YouTube 和 Vimeo 的视频、音频、图像、文字效果、符号和其他元素，并使您的内容引人入胜。强大的预览功能可帮助您在创作时实时观察效果。具体操作步骤请参阅相关章节。

借助 SVG 丰富内容

使用SVG，使您的内容在移动设备，Web 和打印设备上看起来都非常清晰。您可以在SVG 文件中导入并使用多个视觉效果，从而保证最终输出结果拥有最佳用户体验。

智能内容重用

使用变量和代码片段在项目中进行跨主题内容复用可以节省时间和精力。变量或代码段的任何更新都会反映在使用该变量或代码段的地方。可以通过拖放在现有片段中添加一个片段。具体操作，请参阅相关章节。

有条件的内容定义和发布

使用条件标签可创建规则并将其应用于主题、段落和单词。使用条件标签，您可以控制从同一项目生成的不同帮助系统的内容。可以指定并管理条件的前景色和背景色。

整洁的项目结构

RoboHelp 以整洁有序的结构维护所有项目内容。无论是在 RoboHelp 工作区中还是在计算机的资源管理器视图中，您都可以将项目资产（例如样式表，外观和代码片段）组织在单独的文件夹中。

多视图切换

使用屏幕右上角的图标时，可以在作者视图，代码视图和预览视图之间轻松切换。这样可以快速查看内容输出及其 HTML 的预览。

直观的对象属性设置界面

对象属性的设置界面非常直观，在创作内容时，不仅可以简便的调整添的各种对象属性而且可以实时预览。选择添加的对象后，右侧的“属性”面板将显示所选对象的所有属性。调整值并立即查看更改。

强大的皮肤编辑器

自定义的外观可使输出内容具有独特的外观。借助功能强大的 RoboHelp 皮肤编辑器，可以创建更具吸引力和用户友好性的现代化界面。

优化的搜索引擎

通过智能算法为搜索引擎提供一流的搜索结果，并突出显示搜索到的单词。每个输出预设都带有可配置的搜索设置。详情参阅从“生成输出”导航到相关文章。

多格式发布

使用“快速生成”可立即将内容发布为各种输出格式——响应式 HTML5、Microsoft HTML Help (CHM)、WebHelp 和 Mobile App 等。可以一键获取内容为中心的带有 PhoneGap Build许可证的 iOS 和 Android 应用程序。

下一代自适应 HTML5 布局

通过新的响应式 HTML5 布局提供个性化的内容体验、直观的导航以及一流的搜索。通过这种布局可以轻松显示或隐藏诸如词汇表和目录的小部件。详情参阅生成响应式HTML5 输出。

更高的内容可访问性

交付内容以满足最终用户的特殊需求。使用 508-compliant HTML5 布局来创建可发布在多个平台（例如 Web、移动端和印刷品）的内容。

优化的 HTML5 输出

与以前的 RoboHelp 版本相比，现版本加载时间最多可减少 50％，每个主题节省的文件大小约为 20 KB。您还可以通过更快，响应更快的 HTML5 输出体验页面之间的平滑过渡。有关分步帮助，请参阅生成响应式 HTML5 输出。

动态内容过滤器

通过动态内容过滤器提供相关度更高且高度个性化的内容体验。使用条件标签和表达式使目标用户可以轻松过滤内容。详情参阅条件内容配置输出预设相关内容。

添加缩略图

图像可发布为缩略图，并在单击时扩展。这使目标用户更容易在移动设备上查看内容。

一键式移动应用程序生成

一键获取带有 PhoneGap Build 许可证的 iOS 和 Android 内容为中心的移动应用程序。详情参阅参阅生成移动应用程序输出相关内容。

响应式 HTML5 输出中的收藏夹

在自适应 HTML5 输出中，向最终用户提供将主题标记为收藏夹的选项。这样可以节省他们的时间和精力来查找他们经常使用的内容。有关分步帮助，请参阅生成Responive HTML5 输出。

Git 支持

支持 Git ，团队成员能够快速高效地在各种规模的项目上进行协作。

SharePoint Online 支持

使用 SharePoint Online 在整个组织中实现共享和协作，确保每个人都可以编辑同一页面。

 

基本工作流程

RoboHelp 中的基本工作流程：从项目到输出类型


创建项目

一个项目就像一个容器，用于存放所有内容和各种组件，例如目录、摘要、皮肤等，并以构建正确的输出。详情参阅如何创建第一个项目。

AUTHOR THE CONTENT

在 RoboHelp 强大的编辑器中，您可以使用所有需要的元素来创作内容，以使输出引人入胜。创建创作主题并添加文本，图像和多媒体，并使用文本效果、表格、列表等对内容进行格式化。您可以在创作时以所需的布局预览内容。详情参阅使用主题和文件夹。

导入内容

您可以将其他来源的内容（例如 Microsoft Word文件（.docx））导入您的项目。

建立导航

您可以将文本和图像链接到其他内容，并在帮助系统中确定内容层次结构或组织方案。在不同组的内容（主题）之间以及到帮助系统外部的内容（例如 YouTube 视频）之间创建链接。

配置输出内容

你可以在 RoboHelp 中选择输出预设的类型（例如，响应 HTML5 和 PDF），并配置设置以自定义输出的外观和功能。您还可以实施条件性内容以根据最终用户的兴趣，技能水平和其他因素来显示或隐藏内容。

产生输出内容

生成或发布多个平台（例如Web，移动端和印刷品）的输出。可以从各种输出预设中进行选择，例如响应式 HTML5、PDF、Microsoft HTML Help (CHM)、WebHelp、移动应用程序等等。
